amino_acid

class sequal.amino_acid.AminoAcid(value: str, position: int | None = None, mass: float | None = None)[source]

Bases: BaseBlock

Represents an amino acid block that can carry position, modifications, and amino acid value.

Inherits from the BaseBlock class and adds functionality specific to amino acids, such as handling modifications and inferring mass from a predefined dictionary.
