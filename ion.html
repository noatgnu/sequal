ion

class sequal.ion.Ion(seq, charge=1, ion_type=None, fragment_number=None)[source]

Bases: Sequence

Represents an ion fragment sequence object, inheriting properties from the Sequence class.

This class is used to convert a Sequence object into an Ion fragment, with additional properties such as charge, ion type, and fragment number. It also handles modifications and labile groups within the sequence.

Parameters:

  • seq – Sequence The Sequence object to be converted into an Ion fragment.

  • charge – int, optional The charge of the ion (default is 1).

  • ion_type – str, optional The name of the transition type (default is None).

  • fragment_number – int, optional The number of the transition (default is None).

mz_calculate(charge=None, with_water=False, extra_mass=0)[source]

Calculate the mass-to-charge ratio (m/z) of the ion.

Parameters:

  • charge – int, optional The charge of the ion. If not specified, the object’s charge is used.

  • with_water – bool, optional Whether the mass will be calculated with or without water (default is False).

  • extra_mass – float, optional Extra modification of mass that is not represented within the sequence (default is 0).

Returns:

float The calculated m/z value of the ion.
